# Calculator App

A simple command-line calculator built in Python that performs addition, subtraction, multiplication, and division.

## How it works
- Prompts the user to enter two numbers and an operator (+, -, *, /)
- Displays the result
- Loops so the user can keep calculating until they type 'q' to quit

## How to run
1. Make sure Python is installed on your system.
2. Download `calculator.py`.
3. Open a terminal/command prompt in that folder.
4. Run: `python calculator.py`
5. Enter numbers and operators when prompted.

## What I learned
- Writing and calling functions
- Using if/elif/else for decision-making
- Handling user input and basic error cases (like divide by zero)
- Using while loops for repeated actions
